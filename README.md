# BlogBridges

A full-stack blog web application built using Django.

## Features
- User Registration & Login
- Authentication & Authorization
- Create, Update, Delete Posts (CRUD)
- User-specific posts dashboard
- Responsive UI using Bootstrap

## Technologies Used
- Python
- Django
- SQLite
- HTML, CSS, Bootstrap

## Installation
```bash
git clone https://github.com/anushka9733/BlogBridges.git
cd BlogBridges
pip install -r requirements.txt
python manage.py runserver
